Project Profile

ATA CRM v2

Production-oriented CRM for food ingredients trading, built to manage companies, contacts, products, leads, tasks, and controlled email workflows.

Version 2 Next.js 16 Supabase Vercel GitHub Actions
Open Live Demo Open Repository

Problem

Fragmented sales operations

Commercial teams needed one reliable workspace for lead follow-up, profile-level customer data, and role-based collaboration without manual spreadsheets.

Solution

Unified CRM workflow

ATA CRM centralizes pipeline flow, product-company relationships, scoped user permissions, and daily execution tooling in a single interface.

Core Features

What is implemented

  • Lead pipeline with weighted forecast and negotiation outcomes
  • Company, contact, product, and category profile pages
  • Task visibility model with private and shared tasks
  • Manual email with recipient suggestions and analytics scopes
  • Role-based controls for standard, manager, and admin users

Demo Flow

Suggested walkthrough

  1. Open company and contact profiles.
  2. Create a lead, move stages, and finalize win or loss.
  3. Create a linked task and validate visibility.
  4. Send a manual email and inspect personal analytics.
  5. Review dashboard forecast and KPI updates.
