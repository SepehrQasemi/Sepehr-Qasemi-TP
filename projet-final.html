CRM Food Trading - Final Project

Design and delivery of a SaaS CRM for a B2B raw food ingredients trading company.

Business Scenario

Realistic context used for the live demo

The company handles client requests for products such as cocoa, milk powder, and spice blends. The sales cycle requires strict qualification, tracked quotes, and automated follow-up when prospects stay inactive. The CRM centralizes contacts, companies, leads, tasks, and emails to make this process reliable.

Dedicated Workflow and Pipeline

Sales pipeline tailored to business operations

Pipeline

  • New Lead
  • Qualification
  • Sample Sent
  • Quote Sent
  • Negotiation
  • Won
  • Lost

The operational workflow includes task creation, stage movement, history tracking, and a 72-hour automatic follow-up on the "Quote Sent" stage.

Tools Used

Technical stack and delivery workflow
  • Next.js 16 (frontend + API routes)
  • Supabase (PostgreSQL, Auth, RLS)
  • Brevo (transactional emails)
  • Vercel (deployment)
  • GitHub + GitHub Actions (versioning + CI)
