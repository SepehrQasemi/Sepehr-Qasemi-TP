Presentation (8-10 minutes)

Oral demo scenario to present the CRM end-to-end.

Minute-by-minute plan

Short and controlled presentation structure
  • Minute 0-1: business context and CRM goals
  • Minute 1-2: technical architecture (Next.js + Supabase + Brevo + Vercel)
  • Minute 2-4: CRUD demo (company, contact, lead, task)
  • Minute 4-6: pipeline (stage move, quick move, history)
  • Minute 6-7: emails (manual, test, follow-up dry run, task reminders dry run)
  • Minute 7-9: dashboard KPI + funnel + leaderboard + email open/click rates
  • Minute 9-10: bonus demo (CSV/PDF export + BI endpoint), then conclusion

Recommended demo script

Complete path from new lead to Won/Lost
  1. Sign in to the application
  2. Create one company and one contact
  3. Create one product and link it as traded/potential to the company
  4. Create a lead with an estimated value
  5. Move the lead across multiple stages
  6. Create a task, show calendar visibility, and mark it done
  7. Send a test email from a template
  8. Run follow-up dry-run then real execution
  9. Run task reminders dry-run
  10. Review KPI values on the 30-day range, including open/click rates
  11. Export one CSV/PDF report to show bonus delivery quality
